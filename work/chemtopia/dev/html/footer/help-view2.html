Help Public Profile and Setting File Disclosure

Public Profile and Setting File Disclosure

If any user searches your profile in Google, Naver or Yahoo search engine, your available profile may be disclosed. You may correct your public information (on my page), and the personal information that you provided are publicly disclosed, in principle.
Note: Your public profile may be exposed on the search engine.
How to disclose personal information or how to change the disclosed item

1. My Page

  1. You can go to My Page upper side home page, and click the item that you select out of Who can see things on my files section.
  2. The personal file you uploaded may be disclosed in 3 ways such as:
    To all users (all users may access)/only to the followers (only followers of the disclosing user can access)/to no one (only the disclosing user can access).
  3. Default: If you want to disclose your personal information to all users or if you want to hide some information, then you may mark the check or lift check mark. Any changes will be immediately applied. If you load the page again, the changed part will be applied.
  4. Click Upload.

2. Forum

  1. Please define the scope of disclosure in each Topic of the Forum showing user name.
  2. The personal file you uploaded may be disclosed in 2 ways such as:
    To all users (all users may access)/only to the followers (only followers of the disclosing user can access).

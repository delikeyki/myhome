Conversion of
meter per square second (m/s2) to kilometer per square millisecond (km/ms2)

A Unit Conversion Example for Acceleration Quantity

Edit number or edit the unit cell to apply different unit in fixed dimension

Description

The meter per square second (m/s2) or kilometer per square millisecond (km/ms2) represents the quantity of Acceleration. The base dimension of Acceleration is LENGTH per SQUARE TIME. In this unit conversion example, the meter and kilometer belong to the unit of LENGTH, while the second and millisecond are the units of TIME.
